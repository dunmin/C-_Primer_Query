[Ref-github.com/huangmingchuan/](https://github.com/huangmingchuan/Cpp_Primer_Answers/tree/master/ch04)
``` cpp
可以接受。
C++的设计思想是尽可能地“相信”程序员，将效率最大化。然而这种思想却有着潜在的危害，就是无法控制程序员自身引发的错误。因此 Java 的诞生也是必然，Java的思想就是尽可能地“不相信”程序员。
```
